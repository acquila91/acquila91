MEI-Trumpet is an educational web application developed to help music students learn proper fingering and sound production techniques on trumpet.

Features

The app provides customizable exercises where users can specify the metronome value and the desired note range:

Input interface for setting metronome tempo and note range.

Each exercise is structured in a clear 3-step pattern: the first measure is a rest, allowing the user to prepare; the second measure plays a reference note, offering an example to imitate; and the third measure is for the user to play the note. Throughout the exercise, a visual aid is displayed showing the correct trumpet fingering, including adjustments for the third valve slide. This ensures users can easily associate each pitch with the appropriate valve combination.

Other features include the option to add a chromatic tuner and dynamic markings to the exercises, enhancing the learning experience. Users can optionally register via email to track their progress, with performance data collected to help monitor improvement over time.

Research Background

MEI-Trumpet is part of a research project led by Alberto Acquilino at the Schulich School of Music of McGill University, in collaboration with the International Neuroinformatics Coordinating Facility (INCF).
