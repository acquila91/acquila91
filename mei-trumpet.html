MEI-Trumpet is an educational web application designed to facilitate the learning of proper trumpet fingering and sound production techniques. The app offers customizable exercises based on the user's level, allowing them to select the metronome tempo and the desired note range. A 3-measure score is generated in a loop, where the first measure is a rest, the second measure plays the reference note, and the third measure is for the user to play the note.

Features:

  • Custom Exercises: Set your preferred tempo and note range. The app generates a random note within the selected range, providing a structured and repeatable practice session.
  • Trumpet Fingering Visual Aid: A trumpet image displays the correct valve combinations for each note, including slide adjustments for the third valve, guiding users through proper fingering techniques.
  • Chromatic Tuner & Dynamic Indicators: An optional menu allows the user to add a chromatic tuner and include dynamic markings in the exercise, enriching the practice session.

Progress Tracking:

Users can optionally register via email to enable personalized progress tracking. Data on the accuracy of each performance is collected, allowing learners to monitor their improvement over time.

Research Background:

The development of MEI-Trumpet is part of a research project led by Alberto Acquilino at McGill University, in collaboration with the International Neuroinformatics Coordinating Facility (INCF). Ongoing studies are exploring the pedagogical foundations of the app and analyzing its usability and social validity with participants. These insights aim to further refine the app and its impact on trumpet education.

Open-Source and Beta Version:

MEI-Trumpet is available for free as a beta version and can be accessed both as a web application and a mobile app on iOS and Android devices. You can explore the source code in the project's repository here.
